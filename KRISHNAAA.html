Atomic structure and the periodic table

Atomic Number

  • It represents the number of protons in an atom's nucleus
  • Every element has a unique atomic number (e.g., Hydrogen is 1, Helium is 2)
  • The atomic number determines the element's identity and chemical properties
  • It equals the number of electrons in a neutral atom
  • Scientists use the symbol 'Z' to represent atomic number

Important applications:

  • Helps arrange elements in the periodic table
  • Used to predict chemical behaviour
  • Helps calculate mass number (protons + neutrons)

Mass Number

  • Mass number is the total number of protons and neutrons in an atom
  • Represented by the symbol 'A'
  • Mass number = Number of protons + Number of neutrons

Important points:

  • Always a whole number
  • Written as a superscript before the element symbol (e.g., ¹²C)
  • Helps identify different isotopes of an element

Arranging Electrons: Bohr’s model

  • Niels Bohr (1885-1962): Danish physicist who made fundamental contributions to atomic structure and quantum mechanics
  • Won the Nobel Prize in Physics (1922) for his work on atomic structure

Key points of Bohr's Model:

  • Electrons orbit the nucleus in fixed circular paths called shells or energy levels
  • Each shell has a specific energy and can hold a maximum number of electrons
  • Electrons can jump between energy levels by absorbing or emitting energy
  • The first shell can hold 2 electrons, second shell 8, third shell 18, and so on

Electronic Structure

Basic Concept:

  • Electrons orbit the nucleus in shells (energy levels)
  • Each shell has a maximum electron capacity
  • Shells fill from innermost to outermost
